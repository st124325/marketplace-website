# Маркетплейс

Современный маркетплейс с удобным интерфейсом и широким ассортиментом товаров. Проект создан для демонстрации навыков веб-разработки и понимания принципов создания интернет-магазинов.

## Описание проекта

Этот маркетплейс объединяет покупателей и продавцов в одном удобном месте. Сайт включает в себя все основные функции современного интернет-магазина: каталог товаров, фильтрацию, корзину покупок, информацию о категориях и контактную форму.

## Основные возможности

- Каталог товаров с фильтрацией по категориям
- Система поиска товаров
- Корзина покупок с подсчетом количества
- Адаптивный дизайн для всех устройств
- Современный интерфейс с плавными анимациями
- Информационные секции о компании и услугах

## Технологии

Проект создан с использованием чистого HTML, CSS и JavaScript без использования фреймворков. Это позволяет лучше понять основы веб-разработки и создавать быстрые, легкие приложения.

- HTML5 для структуры страницы
- CSS3 для стилизации и анимаций
- Vanilla JavaScript для интерактивности
- Google Fonts для типографики

## Запуск проекта

Проект не требует установки зависимостей. Просто откройте файл `index.html` в браузере.

Для более удобной разработки можно использовать локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

Затем откройте браузер и перейдите по адресу `http://localhost:8000`

## Особенности реализации

### Дизайн

Интерфейс создан с учетом современных принципов UX/UI дизайна. Использованы градиенты, плавные переходы и анимации для создания приятного пользовательского опыта.

### Адаптивность

Сайт полностью адаптирован для работы на мобильных устройствах, планшетах и десктопах. Использованы медиа-запросы и гибкая сетка для корректного отображения на всех экранах.

### Производительность

Код оптимизирован для быстрой загрузки. Минимальное использование внешних библиотек позволяет сайту загружаться быстро даже на медленных соединениях.

## Функциональность

- Фильтрация товаров по категориям (Все, Новинки, Популярные, Со скидкой)
- Добавление товаров в корзину с визуальной обратной связью
- Анимация счетчиков статистики при загрузке страницы
- Плавная прокрутка между секциями
- Адаптивное мобильное меню
